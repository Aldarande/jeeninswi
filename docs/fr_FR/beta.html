Nouveautés beta

En cours — branche beta

  • Nouveau Cache local des images de jeux — téléchargement au premier fetch, stockage dans data/jeeninswi/images/
  • Nouveau Limite quotidienne par jour — le widget affiche la limite configurée pour chaque jour (pas seulement la limite globale)
  • Fix Bouton Rafraîchir — la page se recharge une seule fois après confirmation de fin d'exécution du démon (plus de polling toutes les 3 s)
  • Fix Pochettes de jeux — correction du cas où imageUri retourne un dictionnaire {taille: url} au lieu d'une chaîne
  • Amélioration Action refresh synchrone dans le démon — évite un deadlock sur le verrou asyncio
  • Amélioration Timeout callback Jeedom → démon porté à 30 s pour les actions longues
Feedback Pour signaler un bug ou proposer une amélioration sur la beta, ouvrez une issue sur community.jeedom.com avec le tag JeeNinSwi.

Activer la branche beta

Pour tester la version beta de JeeNinSwi :

  1. Dans Jeedom, allez dans Plugins → Gestion des plugins
  2. Cliquez sur JeeNinSwi
  3. Dans la fiche du plugin, passez la branche de stable à beta
  4. Sauvegardez, puis cliquez sur Mettre à jour
  5. Redémarrez le démon depuis la page de configuration du plugin
Avant de basculer en beta Pensez à noter la version stable actuelle afin de pouvoir revenir en arrière si nécessaire. Les dépendances peuvent changer entre versions — relancez l'installation des dépendances après chaque mise à jour beta.

Revenir à la version stable

Si la beta pose problème, revenez en stable en quelques secondes :

  1. Plugins → Gestion des plugins → JeeNinSwi
  2. Passez la branche de beta à stable
  3. Sauvegardez, puis Mettre à jour
  4. Réinstallez les dépendances et redémarrez le démon
